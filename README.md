# Snake-Game 
(HTML + CSS + JavaScript)

 Game Features
- Player Name input
- Choose Snake Speed: Slow, Medium, Fast
- Select Theme (Aesthetic Sky & Pink, etc.)
- Snake movement { Arrow Keys + WASD }
- Spacebar to Pause / Resume
- Food Spawning
- Score + High Score
- High Score Reset
- Start Game / Pause / Restart / New Game
- Current Score Display

 Extra Features
- Beautiful modern UI
- Smooth snake movement
- Fully responsive gameplay
- Play again without reloading page

 How to Play
- Use arrow keys or **W A S D** to move the snake
- Press **Spacebar** to Start or Pause
- Eat food to increase score and grow
- Avoid hitting the wall & your own tail
- Open `index.html` in your browser to play
